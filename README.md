 🎓 Student Score Prediction

This project is a simple Machine Learning model that predicts student scores based on the number of hours studied.
---
📌 Overview
The model uses Linear Regression to understand the relationship between study hours and exam scores, and predicts the score for new input values.
---
🚀 Features
- Predicts student score based on study hours
- Simple and beginner-friendly ML project
- Uses basic data for training
- Command-line based input
---
🛠 Technologies Used
- Python  
- NumPy  
- Scikit-learn  
---

 📊 How It Works
1. Input data: Study hours and corresponding scores  
2. Train model using Linear Regression  
3. User enters study hours  
4. Model predicts the score
